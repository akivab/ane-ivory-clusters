Identifying Regional Design Templates of Ancient Near Eastern Ivory Sculptures of Women Using Computer Technology

What's this all about?

We cluster images based on features deatiled by art historians using well known clustering algorithms

The result is an interesting set of clusters that may help us better understand the similarities and differences between ANE sculptures
